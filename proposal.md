## Description
A 20-second looping animation featuring a gradient blue sky background, randomly falling snow, and a rolling snowball.

## Features
- Feature 1
	- Use the drawing function of tutle to create grading blue sky, a little bit complex snow pattern, an irregularly shaped snowball.
- Feature 2
	- Use random function on size, speed, locations.
- Feature 3
	- Video recording funtion to combine all parts and get the final animation.

## Challenges
- Use tutle to draw grading blue sky, a little bit complex snow pattern, an irregularly shaped snowball.
- How to Seperate layers of sky, snow, and snow ball to create, combine together at final.

## Outcomes
Ideal Outcome:
- A gradient effect blue sky, randomly and realistically falling snow, snowball moving at varying speeds, ultimately combining these parts into a looping animation.

Minimal Viable Outcome:
- Blue sky, snow vary in size but maintain a constant speed, snowball movable, a complete looping animation.
